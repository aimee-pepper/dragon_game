# Dragon Specialty Combos Reference

Special name overrides triggered by specific color + finish or finish + element combinations. These replace the default compound naming system when conditions are met.

When multiple combos match, use this priority:
1. Color + Finish specialty name (replaces color name entirely, e.g. "Ruby")
2. Finish + Element modifier prefix (applied before color name, e.g. "Molten Red")
